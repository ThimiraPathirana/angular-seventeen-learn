structural works!

If directive

  • With old ngIf syntax
    • A is grater than B
    B is grater than A
  • With new if syntax
    • @if(a>b) { {{a}} is grater than {{b}} } @else { {{b}} is grater than {{a}} }

    For directive

  • With old ngFor syntax
    • {{user}}
  • With new for syntax
    • @for(user of users; track user) {
    {{user}}
    }

    Switch directive

  • With new switch case syntax
    • @switch(role) { @case('admin') {

    Admin user role

    } @case('manager') {

    Manager user role

    } @default {

    Default user role

    } }
